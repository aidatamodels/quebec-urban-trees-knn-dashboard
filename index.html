À propos de l’application

Objectif

Ce dashboard permet d’explorer les arbres répertoriés de la Ville de Québec et d’identifier, à l’aide de l’algorithme K-Nearest Neighbors, les arbres les plus similaires à un arbre sélectionné.

Comment l’utiliser

  • Utilisez les filtres du panneau de gauche.
  • Les filtres s’appliquent automatiquement.
  • Zoomez et déplacez-vous sur la carte.
  • Survolez un arbre pour obtenir un aperçu rapide sur desktop.
  • Touchez ou cliquez sur un arbre pour le sélectionner comme référence.
  • Consultez les arbres similaires dans le panneau du bas.

Rôle de K-NN

K-NN compare l’arbre sélectionné aux autres arbres à partir de plusieurs caractéristiques : diamètre, type d’arbre, type de lieu, type de propriété, coordonnées géographiques et autres variables disponibles.

K-NN similarité données ouvertes géospatial

Distance K-NN

La distance K-NN est un score de similarité. Elle ne correspond pas directement à une distance en kilomètres. Plus cette distance est faible, plus l’arbre voisin est proche du profil de l’arbre sélectionné.

Jeu de données

Les données proviennent du portail Données Québec, à partir du jeu de données des arbres répertoriés de la Ville de Québec, accessible par API CKAN.

Finalité pédagogique

Ce projet illustre une utilisation concrète de la science des données, de l’exploration cartographique et de l’apprentissage automatique dans une application web interactive.
